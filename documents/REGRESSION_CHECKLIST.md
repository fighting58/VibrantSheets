# Regression Checklist

목표: 핵심 시나리오 변경 후 품질 회귀를 빠르게 점검합니다.

## A. 기본 편집/선택
- 단일 셀 클릭/키보드 이동 정상
- 드래그 다중 선택 정상
- 병합 셀 선택 시 앵커로 이동
- 더블 클릭 편집 진입 정상
- IME 입력(한글 조합) 첫 글자 손실 없음

## B. 행/열 자동 확장
- 마지막 행에서 `Enter` → 행 자동 확장
- 마지막 열에서 `Tab`/`→` → 열 자동 확장
- 붙여넣기 범위가 현재 그리드를 넘을 때 자동 확장

## C. 병합/테두리
- 병합 생성/해제 정상
- 병합 포함 범위 복사/붙여넣기 시 병합 유지
- 병합 경계 테두리 우선순위: 마지막 적용이 우선
- 얇은선(1px) 테두리 상단/좌측 누락 없음

## D. 스타일/서식
- 배경색, 글꼴 굵기/기울임/밑줄/취소선 유지
- 숫자 서식: 일반/퍼센트/날짜 정상 표시
- 통화 서식: KRW/USD 구분 유지

## E. 파일 입출력
- VSHT 저장/불러오기 정상
- CSV 저장/불러오기 정상 (구분자 감지 포함)
- XLSX 저장/불러오기 정상
  - 병합 유지
  - 배경/테두리 유지
  - `styles.xml` 복구 경고 없음

## F. 인쇄
- 인쇄 영역 미지정 시 경고 동작
- 인쇄 영역 지정 후 미리보기 정상
- 헤더/푸터 슬롯 토글 정상
- 에디터 오버레이(선택 하이라이트 등) 인쇄에 포함되지 않음
- 페이지 분할/여백/방향 반영 정상

## G. 기타
- Undo/Redo(있다면) 정상
- 검색/바꾸기 정상
