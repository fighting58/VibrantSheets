# Regression Checklist

## 인쇄 (Phase 12)
- [ ] 인쇄영역 미설정 상태에서 인쇄 버튼 클릭 시 에러 메시지 노출
- [ ] 인쇄영역 설정/해제 토글 동작 및 툴팁 표시
- [ ] 페이지 브레이크 프리뷰와 인쇄 미리보기의 페이지 분할 범위 일치
- [ ] Scale 100% 기준에서 다중 페이지가 정상 생성되는지 확인
- [ ] 커스텀 인쇄 페이지 렌더링 시 1페이지가 비지 않는지 확인
- [ ] 헤더/푸터 옵션(좌/중/우) 설정이 반영되는지 확인
- [ ] 선택 영역 인쇄 시 오버레이 하이라이트가 인쇄물에 포함되지 않는지 확인

## 그리드/선택/편집
- [ ] 값이 있는 셀 드래그 시 멀티 셀 선택 정상
- [ ] 병합 셀 선택 후 입력 시 값 덮어쓰기 정상
- [ ] 행/열 자동 확장(붙여넣기/이동/입력) 동작
- [ ] 컬럼 헤더가 `AA`, `AB` 이후로 정상 표시

## 테두리/서식
- [ ] 병합 셀 경계에서 얇은선, dot/dash 우선순위 정상
- [ ] 테두리 충돌 시 “나중에 설정한 테두리” 우선 적용
- [ ] 배경색/폰트/정렬 등 기본 서식 유지

## 파일 입출력 (XLSX/CSV/VSHT)
- [ ] 병합 범위 저장/불러오기 유지
- [ ] strikethrough 저장/불러오기 유지
- [ ] 통화 포맷(KRW/USD) 재저장 시 단위 유지
- [ ] 배경색 포함 서식 반영 여부 확인
